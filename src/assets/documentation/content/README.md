# உள்ளடக்க அட்டவணை - தினசரி திருக்குறள் ஆவணங்கள்

இந்தக் கோப்பகத்தில் "தினசரி திருக்குறள்" இணையதளத்திற்கான முழுமையான தமிழ் உள்ளடக்கங்கள் சேமிக்கப்பட்டுள்ளன. ஒவ்வொரு கோப்பும் ஒரு குறிப்பிட்ட பக்கத்திற்கான விரிவான தகவல்களைத் தூய தமிழ் நடையில் கொண்டுள்ளது.

## உள்ளடக்கங்கள்

1.  [முகப்பு பக்கம் (01_home.md)](file:///01_home.md)
    - வரவேற்பு செய்தி, அன்றாட வாழ்வின் ஒழுக்கம் மற்றும் அடிப்படை அறிமுகம்.
2.  [நாளின் குறள் (02_daily_kural.md)](file:///02_daily_kural.md)
    - குறள் விளக்கம், திருவள்ளுவர் வரலாறு மற்றும் நவீன காலப் பொருத்தம்.
3.  [குறள் தேடல் (03_kural_search.md)](file:///03_kural_search.md)
    - 133 அதிகாரங்கள் வகைப்பாடு, தேடல் கருவிகள் மற்றும் வடிகட்டும் முறைகள்.
4.  [சேமித்த குறள்கள் (04_saved_kurals.md)](file:///04_saved_kurals.md)
    - சேமிப்பதன் நன்மைகள் மற்றும் பகிர்தல் வசதிகள் பற்றிய விளக்கம்.
5.  [நம்மைப் பற்றி (05_about_us.md)](file:///05_about_us.md)
    - எமது நோக்கம், குழு அறிமுகம் மற்றும் கலாச்சார அர்ப்பணிப்பு.
6.  [அம்சங்கள் (06_features.md)](file:///06_features.md)
    - செயலியில் உள்ள தொழில்நுட்ப வசதிகள் மற்றும் புள்ளிவிபர வரைபடங்கள் பற்றிய விவரம்.
7.  [கீழொட்டு உள்ளடக்கம் (07_footer.md)](file:///07_footer.md)
    - காப்புரிமை, பதிப்புரிமை மற்றும் முக்கிய இணைப்புகள்.

## தரக் கட்டுப்பாடுகள்

- **தூய தமிழ்**: எங்கும் ஆங்கிலச் சொற்கள் பயன்படுத்தப்படவில்லை.
- **ஆழமான விளக்கம்**: ஒவ்வொரு பகுதியும் குறைந்தது 250 வார்த்தைகளைக் கொண்டுள்ளது.
- **இலக்கிய நடை**: திருக்குறளின் மேன்மைக்கு ஏற்றவாறு செம்மையான தமிழ் நடை கடைப்பிடிக்கப்பட்டுள்ளது.

இந்த ஆவணங்கள் அனைத்தும் எமது இணையதளத்தின் உள்ளடக்கத்தை மேன்மையுறச் செய்யும் நோக்கில் உருவாக்கப்பட்டவை.
